# hi~

欢迎来到 PinkLine

PinkLine 是一款基于 Nuxt.js 的个人博客模版, 其快速, 简约, 轻量.

[在线演示]() ｜ [使用说明](/article)

## 特色

- 基于 Nuxt.js
- 强大的CMS: @nuxt/content
- SEO友好
- 响应式布局
- 明暗模式
- 文章搜索
- 评论系统

## 部署

本地部署

``` bash
git clone https://github.com/sooooooooooooooooootheby/PinkLine

cd PinkLine

npm i

npm run dev
```
